Wisdawn is a collaborative peer-led learning ecosystem. Shifting away from traditional teacher-centric models, Wisdawn empowers students to host mini online workshops, share knowledge, and solve academic doubts collaboratively through a gamified engagement system.

## 🛠 Tech Stack & Architecture

The web platform is built with a high-performance, modern frontend stack designed for real-time collaboration and responsive design.

### Frontend Core & UI

* **React & TypeScript:** For building a type-safe, component-based user interface.
* **Tailwind CSS:** A utility-first CSS framework for rapid and consistent styling.
* **Framer Motion:** Powering smooth, high-quality animations and transitions.
* **Lucide React:** A clean and consistent iconography set.
* **@fontsource/inter:** Premium typography for enhanced readability.

### Backend & Integration

* **Firebase:** Handles real-time database synchronization, cloud storage for resources, and secure user authentication.
* **Clsx & Tailwind-merge:** Utilities for managing dynamic CSS classes efficiently.

### Dev/Build Tooling

* **Vite:** A lightning-fast build tool and development server.
* **ESLint:** Ensuring code quality and adherence to React best practices.
* **TypeScript-ESLint:** Advanced linting for TypeScript-specific patterns.

## ✨ Features

* **Mini Online Workshops:** Create topic-based sessions and upload learning resources directly from your browser.
* **Peer Doubt Solving:** A dedicated system allowing students to post questions and receive collaborative responses in real-time.
* **Point-Based Mentoring:** Actively encourages peer tutoring and knowledge sharing through a gamified pairing system.
* **Gamified Progress Tracking:** Rewards participation and contribution to foster a community-driven academic environment.
* **Cross-Platform Sync:** Start a workshop on the Android app and finish it on the Web without losing progress.

## 🚀 Getting Started

### Prerequisites

* **Node.js** (v18.0.0 or higher)
* **npm** or **yarn**

### Installation

1. Clone the repository:
```bash
git clone https://github.com/rstdax/wisdawn-website.git

```


2. Install dependencies:
```bash
npm install

```


3. Configure your Firebase environment variables in a `.env` file.
4. Start the development server:
```bash
npm run dev

```
